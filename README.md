# Car-Detailing
Car-Detailing project for Frontend Design Techniques

# Technologies
## Backend
- .NET 6

## Frontend
- React
- TypeScipt
- TanStack Query V4

# How to run
## Frontend
### Without docker
First it is needed to install all required packages. Go to `/frontend` and run
```bash
> npm install
```
Then write
```bash
> npm run dev
```
